# ssh-bot
multiple connect ssh run command line

## Configuretion
create file [`host.txt`](./host.txt) for list ssh ip
create file [`pass.txt`](./host.txt) for list ssh password

## Installation

```bash
git clone https://github.com/krittakondev/ssh-bot.git
cd ssh-bot
go get
```

## Run
```
go run main.go
```
